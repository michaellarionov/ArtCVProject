================================================================================
STREAMLIT UI CUSTOMIZATION GUIDE
================================================================================

This guide explains how to customize the default Streamlit UI to match your
own design ideas. No AI assistance needed - just edit the files!


================================================================================
PART 1: THEME COLORS (.streamlit/config.toml)
================================================================================

Location: .streamlit/config.toml

The easiest way to change the look of your app. Edit these values:

    [theme]
    primaryColor = "#6366f1"              # Buttons, links, accents
    backgroundColor = "#0f0f0f"           # Main content background
    secondaryBackgroundColor = "#1a1a1a"  # Sidebar background
    textColor = "#fafafa"                 # All text
    font = "sans serif"                   # "sans serif", "serif", "monospace"

COLOR IDEAS:
    Dark mode (current):     primaryColor="#6366f1", backgroundColor="#0f0f0f"
    Light mode:              primaryColor="#6366f1", backgroundColor="#ffffff"
    Warm/earthy:             primaryColor="#d97706", backgroundColor="#1c1917"
    Ocean blue:              primaryColor="#0ea5e9", backgroundColor="#0c1929"
    Forest green:            primaryColor="#22c55e", backgroundColor="#0a0f0a"

TIP: Use https://coolors.co to generate color palettes

================================================================================
PART 3: LAYOUT OPTIONS
================================================================================

COLUMNS - Side by side elements:
    col1, col2 = st.columns(2)           # Two equal columns
    col1, col2 = st.columns([2, 1])      # Left column twice as wide
    
    with col1:
        st.write("Left content")
    with col2:
        st.write("Right content")

SIDEBAR - Side panel:
    with st.sidebar:
        st.header("Controls")
        # Your widgets here

TABS - Tabbed sections:
    tab1, tab2 = st.tabs(["Generate", "Gallery"])
    
    with tab1:
        st.write("Generation controls")
    with tab2:
        st.write("Previous generations")

EXPANDER - Collapsible sections:
    with st.expander("Advanced Settings"):
        # Hidden by default, click to expand
        st.slider("Steps", 20, 100, 50)

CONTAINER - Group elements:
    with st.container():
        st.write("Grouped content")

PART 5: CUSTOM CSS (Advanced)
================================================================================

Inject custom CSS to override Streamlit's default styles:

    st.markdown("""
    <style>
        /* Hide Streamlit branding */
        #MainMenu {visibility: hidden;}
        footer {visibility: hidden;}
        
        /* Custom button style */
        .stButton > button {
            background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
            border: none;
            border-radius: 12px;
            padding: 12px 24px;
            font-weight: bold;
        }
        
        /* Custom sidebar width */
        [data-testid="stSidebar"] {
            width: 350px !important;
        }
        
        /* Custom header style */
        h1 {
            color: #6366f1;
            font-family: 'Georgia', serif;
        }
        
        /* Add background image */
        .stApp {
            background-image: url("https://example.com/bg.jpg");
            background-size: cover;
        }
    </style>
    """, unsafe_allow_html=True)

PART 7: RESTRUCTURING THE LAYOUT
================================================================================

Example: Move controls to main area instead of sidebar:

    # Instead of:
    with st.sidebar:
        selected_period = st.selectbox(...)
    
    # Do this:
    col1, col2 = st.columns([1, 2])
    
    with col1:
        st.header("Settings")
        selected_period = st.selectbox(...)
        selected_palette = st.selectbox(...)
        generate_btn = st.button("Generate")
    
    with col2:
        st.header("Output")
        if generate_btn:
            st.image(generated_image)

================================================================================
TIPS FOR DESIGNING YOUR OWN UI
================================================================================

1. SKETCH FIRST
   Draw your layout on paper before coding. Decide:
   - Where do controls go? (sidebar vs main area)
   - What sections do you need?
   - What's the visual hierarchy?

2. START SIMPLE
   Get basic functionality working, then add styling.

3. USE THE DOCS
   The Streamlit docs have live examples you can copy.

4. ITERATE
   Make small changes, see the result, repeat.

5. INSPECT ELEMENTS
   Use browser DevTools (F12) to find CSS class names for custom styling.
